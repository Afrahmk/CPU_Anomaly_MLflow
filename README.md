# Détection d’anomalies CPU avec MLflow

## Description
Ce projet utilise **Python** et **MLflow** pour détecter les anomalies dans l’utilisation CPU.  
Le modèle **IsolationForest** est utilisé pour identifier les pics d’utilisation CPU.  
Les résultats (métriques et modèles) sont enregistrés dans **MLflow** pour suivi et visualisation.

## Fonctionnalités
- Simulation de données CPU normales et anomalies
- Détection automatique des anomalies
- Sauvegarde des métriques et modèles dans MLflow
- Visualisation graphique des anomalies (courbes CPU)
